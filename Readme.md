# Wikipedia Mobile Automation Project

## Overview
This project automates key functionalities of the Wikipedia Android app using **Appium** and **TestNG**.  
It demonstrates mobile automation skills including element identification, interactions, and reporting.

## Tools & Technologies
- Appium (UiAutomator2)
- Java 17
- TestNG
- Eclipse IDE
- Android Real Device
- Android Studio & ADB

## Features Automated
1. Launch Wikipedia App  
2. Click on Contents section  
3. Search for a term (e.g., "India")  
4. Click the first search result  
5. Validate page navigation
